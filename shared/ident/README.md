# /shared/ident — the studio bumper

The shared **boot moment** that grounds every cabinet back to the arcade. Per the
share-by-identity rule (`CLAUDE.md` §6), the full entrance experience composes:

**`shared/ident` (studio bumper) → `games/<game>/attract` (game demo) → `play`.**

## Files
- `bumper.html` — the studio bumper asset (relocated in the migration from the
  flat-repo `What If Arcade - Boot (Button).html`). **Currently standalone /
  unwired** — nothing references it yet. Mounting the bumper → attract → play flow
  is the IRONLINE game-pod's job (see the migration kickoff, "After migration").
